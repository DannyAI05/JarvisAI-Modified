- Voice input
- Text-based chat
- Text-to-speech responses

## Technologies Used

- Python
- Tkinter
- SpeechRecognition
- Edge-TTS
- Pillow
- PyInstaller

## Installation

1. Download JarvisAI_Setup.exe
2. Run the installer
3. Launch JarvisAI

## Usage

1. Click Speak to use voice commands.
2. Type messages into the chat box.
3. Click Send to interact with JarvisAI.
