# CHANGELOG

All notable changes to the JarvisAI project will be documented in this file.

## [1.0.0] - June 2026

### Added

* Developed a desktop-based AI virtual assistant using Python.
* Implemented a graphical user interface (GUI) using Tkinter.
* Added text-based communication between the user and JarvisAI.
* Added voice command functionality using SpeechRecognition.
* Implemented text-to-speech responses using Edge-TTS.
* Added the ability to open websites and desktop applications through commands.
* Integrated chat display and input field for user interaction.
* Added custom application logo and icon support.
* Created a standalone executable file (JarvisAI_Setup.exe) using PyInstaller.
* Created a project preview webpage for software distribution.
* Added GitHub repository and release management support.

### Fixed

* Resolved missing button display issues in the GUI.
* Fixed image loading and resource path errors.
* Corrected layout and frame management problems in Tkinter.
* Resolved executable build issues during packaging.
* Fixed file path handling for application resources.

### Improved

* Improved user interface appearance and responsiveness.
* Enhanced command processing and response generation.
* Improved software deployment and distribution methods.
* Organized project documentation and project structure.
* Enhanced overall user experience and application stability.

### Known Limitations

* Requires an internet connection for some speech-related functionalities.
* Limited natural language understanding compared to large AI models.
* Currently available only as a Windows desktop application.

### Future Plans

* Integrate ChatGPT API for advanced conversations.
* Add offline speech recognition capabilities.
* Add machine learning and personalization features.
* Develop mobile and cross-platform versions.
* Implement cloud synchronization and user profiles.
* Add multilingual support and custom voice profiles.
